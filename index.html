What you're going to learn from this club ?



Web Development

In this course first you going to learn the basics things behind Web Development i.e, HTML, CSS, & JavaScript using these things you going to build some web application, websites etc. Further you going to learn the backened part of the website i.e, like JQuery, Django, etc.

App Development

In this course you'll going to learn how to build naive application using Flutter which is kindly trending in current market, how to use or call API's in your apps, connected with the DBs or how to published it on Play Store.

Game Development

In this course you'll going to learn how to build interactive games using Unity Engine which works with scripting language i.e, JavaScript/C# & how to programming the games, build the structures, or using assest in your game from the assest store & how to test it on your own device. Further you going to learn glimpse of the AR/VR technolgy which is trending currently in the market.

Competitive Programming

Competitive Programming is the most important thing in your B.Tech curriculum which helps you to understand the concepts of Data Structure & Algorrithm better by coding them on actual problems. This course going to help you to clear the interview of good product based companies like Amazon, Google, Uber, Ola, Oyo etc
